# School Semester Project

A simple web application built as part of a school assignment over a two-week period.

The project was developed in a short time window and focuses on understanding basic web development concepts using PHP and vanilla frontend technologies.

---

## 🛠️ Stack

- PHP (backend logic)
- HTML5
- CSS3
- JavaScript (vanilla)

---

## 🎯 Purpose

The goal of the project was to practice:
- server-side logic in PHP
- working with forms and requests
- basic frontend structure and interactivity
- structuring a simple full-stack web application

---

## 📌 Notes

This project was created for educational purposes and is not intended for production use.
